### 2. OpenAlex Institution Claim
**Why:** OpenAlex is the open replacement for Microsoft Academic, used by many discovery tools.

**Action:**
1. Check if The Unjournal exists: https://openalex.org/institutions?search=unjournal
2. If exists, verify it links to your ROR
3. If not, it should auto-populate from ROR within weeks
4. Contact OpenAlex if corrections needed: team@ourresearch.org

**Impact:** All Unjournal publications get properly affiliated in OpenAlex

### 3. ORCID Organization Portal
**Why:** Link team members' ORCIDs to The Unjournal as an employer/affiliation.

**Action:**
1. Each team member with ORCID:

   - Add Employment or Affiliation

2. Consider ORCID Member organization (paid) for more control

**Impact:** Team credibility, author disambiguation, organizational visibility
